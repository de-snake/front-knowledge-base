# Asset investment diligence methodology

This folder now has one canonical research methodology source:

- [`research-composition-methodology.md`](research-composition-methodology.md)

Use that page for current asset / platform / product diligence work. It defines the latest model:

```text
asset baseline + platform baseline + product delta = composed opportunity view
```

It also defines the researcher / formatter split:

- researchers create reusable source-backed evidence artifacts;
- formatters compose the requested memo, UI summary, public report, or other presentation from those artifacts;
- formatter output must not be the only place where a material source fact exists.

Older standalone research-methodology pages have been removed. Runner-contract files that still exist in this directory are compatibility support for existing workflow tooling and reproducible packages; do not treat them as separate research methodology sources.
