Deployments

Smart contracts​

Mainnet​

The protocol is currently deployed on Ethereum, Base, Arbitrum, Avalanche, Plume, Binance Smart Chain, Optimism, HyperEVM, Monad and Pharos.

The deployed code commit for v3.1.0 is 6b9d36eabee48728486f377ea2766a5cd233c555. This commit has been reviewed by 4 independent security reviews. The deployment of the protocol has been verified by Sherlock in this report.
